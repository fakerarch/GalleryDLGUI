Sleep / Rate Limiting
Delays between requests to avoid rate limits and bans
Delay timings (seconds)
Sleep between downloads--sleep — wait N seconds before each file download
Sleep between requests--sleep-request — between each HTTP request during extraction

Use ranges like 1-3 to randomize sleep duration. gallery-dl's default sleep varies by site — many sites (Instagram, e621, Danbooru) already have built-in delays.

Download Archive
Track downloaded files to avoid re-downloading
Archive database
Archive file path --download-archive — SQLite DB storing downloaded file IDs. Downloads already in the archive are skipped.
The archive is a SQLite3 database. Each downloaded file's ID is stored and checked before future downloads — even if you rename or move files, they won't be re-downloaded. Use {category}.sqlite3 in the path to create per-site archives.

Authentication
Credentials applied to every download
Credentials are passed as -u / -p flags on every download. Required for nijie; optional for danbooru, e621, inkbunny, mangadex, subscribestar, and others.
Login credentials
